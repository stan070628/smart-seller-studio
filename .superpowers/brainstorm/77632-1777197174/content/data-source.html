쿠팡 광고 데이터를 어떻게 가져올까요?

보고서 기능의 핵심 아키텍처 질문 — 쿠팡 광고 관리자(advertising.coupang.com)는 공개 API가 없습니다

A

CSV 업로드

쿠팡 광고 관리자에서 직접 CSV를 내려받아 앱에 업로드

장점

  • 구현 가장 간단
  • 데이터 100% 정확
  • 쿠팡 정책 위반 없음

단점

  • 매번 수동으로 내려받아야 함
  • 실시간 데이터 불가
B

수동 입력

노출수, CTR, ROAS 등 핵심 지표를 사용자가 직접 입력

장점

  • 구현 최소화
  • 필요한 지표만 선택 가능

단점

  • 입력 번거로움
  • 실수 가능성
  • 많은 상품일 때 비현실적
C

브라우저 자동화 (스크래핑)

앱 서버에서 Puppeteer/Playwright로 쿠팡 광고 관리자 자동 접속

장점

  • 자동화 — 클릭 한 번으로 최신 데이터
  • 정기 갱신 가능

단점

  • 쿠팡 로그인 세션 관리 복잡
  • UI 변경 시 깨질 위험
  • Vercel 환경에서 Playwright 제약
D

쿠팡 광고 파트너 API

공식 파트너 프로그램 신청 후 API 키로 데이터 조회

장점

  • 공식적이고 안정적
  • 자동화 완전히 가능

단점

  • 파트너 신청 필요 (승인까지 시간)
  • 현재 파트너 권한 있는지 불명

💡 추천: A (CSV 업로드) — 빠른 구현, 데이터 정확성, 쿠팡 정책 안전
