A

Claude Code 서브에이전트

.claude/agents/coupang-report.md 파일 생성 → Claude에게 "보고서 생성해줘"라고 말하면 자동 실행

호출 방법
# 이 프로젝트 안에서 Claude에게 말하면 됨
"쿠팡 광고 보고서 생성해줘"
# → Claude가 자동으로 에이전트 호출
# → gstack 브라우저로 두 사이트 로그인
# → 데이터 수집 → Opus로 분석 → PDF 생성

장점

  • Claude Code에 완전히 통합됨
  • 별도 터미널 명령 불필요
  • 대화형으로 추가 분석 요청 가능

단점

  • Claude Code 세션 안에서만 호출
B

gstack 커스텀 스킬

~/.claude/skills/coupang-report/SKILL.md 설치 → /coupang-report 명령으로 호출

호출 방법
# 어느 프로젝트에서든 슬래시 명령
/coupang-report
# CLAUDE.md에 라우팅 규칙 추가 필요

장점

  • 프로젝트 무관하게 어디서나 사용
  • gstack 에코시스템과 통합

단점

  • gstack 스킬 구조 학습 필요
  • 이 프로젝트에만 쓸 거라면 과도함
C

셸 스크립트 + npm 명령

scripts/coupang-report.sh + package.json script 등록

호출 방법
# 터미널에서 직접 실행
npm run report
# 또는
bash scripts/coupang-report.sh

장점

  • Claude 없이도 독립 실행 가능
  • cron으로 자동화 가능

단점

  • Claude 분석 없이 단순 스크래핑만
  • 에러 대응이 어려움
