실행 흐름

1. 요청 파싱
"7일 보고서 생성해줘" → 기간 추출 (없으면 30일 기본값)
2. Ads 접속
gstack chromium-profile 쿠키로 자동 로그인 시도 → 실패 시 수동 로그인 안내
3. Ads 수집
노출·클릭·CTR·주문·CVR·광고비·매출·ROAS, 캠페인별 상세
4. Wing 접속
같은 계정이면 자동 로그인 → 실패 시 수동 안내
5. Wing 수집
상품 목록(재고·이미지·상태), 최근 리뷰 현황
6. Opus 분석
펀넬 분석 · 업계 평균 비교 · 원인 진단 · 액션플랜 생성
7. 보고서 생성
HTML → 브라우저 미리보기 → PDF → 바탕화면 저장
