A. 체크박스 + 하단 버튼

각 행에 체크박스 추가. 선택 시 테이블 하단에 "N개 상품 대량등록" 버튼 활성화. 버튼 클릭 → 등록탭으로 이동하며 번호 자동 입력.

장점

  • 복수 선택이 명확
  • 실수가 적음

단점

  • 체크박스 컬럼 공간 필요
  • 기존 테이블 레이아웃 변경

세라믹 머그컵 350ml 71점
보냉 도시락통 2단 92점

B. 행마다 "등록" 버튼

각 행 오른쪽에 "등록" 버튼 추가. 클릭 시 해당 상품번호를 대량등록 큐에 추가. 버튼을 여러 번 눌러서 복수 추가 후 등록탭 이동.

장점

  • 직관적이고 간단
  • 기존 레이아웃 최소 변경

단점

  • 여러 개 추가 시 번거로움
  • 몇 개 추가했는지 파악 어려움

스테인리스 텀블러 500ml 87점
세라믹 머그컵 350ml 71점
3 상품 대량등록 큐에 추가됨

C. 체크박스 + 고정 액션 바

체크박스 선택 시 테이블 하단에 항상 보이는 고정 바 활성화. "N개 큐에 추가됨" 표시 + "등록탭으로 이동" 버튼. 추가하면서 계속 탐색 가능.

장점

  • 선택 상태가 항상 보임
  • 탐색 중에도 누적 추가 가능

단점

  • A와 구현 복잡도 유사
