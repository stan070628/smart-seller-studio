## 목표

3개월(12주) 내 온라인 쇼핑몰 월 매출 1,000만원 달성

- **채널**: 네이버 스마트스토어(메인) + 쿠팡(보조)

## 전략 구조: C형 복합

레드오션 회피 + 데이터 기반 진입 + 남성 니치 소규모 테스트 병행

### ① 메인 70개 — 도매꾹 위탁 (아이템스카우트 기반)

**선정 기준 (아이템스카우트 필터)**

| 기준 | 수치 |
|---|---|
| 월 검색량 | 3,000 ~ 30,000 |
| 경쟁 상품 수 | 500개 미만 |
| 상위 상품 리뷰 수 | 50개 미만 |
| 가격대 | 8,000원 ~ 50,000원 |
| 마진 | 30% 이상 |
| 도매꾹 위탁 조건 | 배송 3일 이내, 위탁 허용 |

**실행 방법**
- 조건 충족 키워드 15개 발굴
- 각 키워드당 도매꾹 매칭 상품 4~5개 선별
- 총 70개 등록 (Week 2 완료 목표)
- 카테고리 무관 — 데이터가 나오는 곳이면 어디든

### ② 보조 10개 — 남성 니치 테스트 (직장인 데스크 셋업)

**선정 배경**
- 스마트스토어/쿠팡은 여성 중심 → 남성 타겟은 상대적 블루오션
- 단, "남성" 자체는 니치가 아님 — "남성 + 직장인 데스크 셋업" 교차점 공략
- 연중 수요 일정, 소형 상품, 브랜드 로열티 낮음

**상품 예시**
- USB 케이블 정리 클립/홀더
- 모니터 받침대 (원목, 알루미늄 소재)
- 특수 소재 마우스패드 (XXL, 속건 등)
- 노트북 거치대 (접이식, 알루미늄)
- 데스크 수납 트레이

**판단 기준**
- Week 4에 CTR / 전환율 데이터 검토
- 성과 없으면 전량 교체, 성과 있으면 확대

### ③ 코스트코 병행 — 2주에 1회 방문

**현장 즉시 검증 루틴**

```
상품 발견
   ↓
네이버/쿠팡에서 상품명 검색
   ↓
셀러 3명 미만 AND 온라인 최저가 ÷ 코스트코 가격 > 1.4배
   → 진입 후보로 메모 (상품명 + 바코드 + 두 가격)
   ↓
셀러 3명 이상 OR 마진 40% 미만
   → 패스
```

**사입 프로세스**

| 단계 | 수량 | 조건 |
|---|---|---|
| 첫 테스트 사입 | 1~2개 | 등록 후 클릭 발생 여부 확인 |
| 추가 발주 | 5~10개 | 구매 1건 이상 발생 확인 후 |
| 스케일 사입 | 20개+ | ROAS 300% 이상 + 리뷰 5개 이상 |

**주의사항**
- 코스트코 시즌 한정 상품 제외 (재입고 불투명)
- 부피/무게 큰 상품은 보관 공간 먼저 확인
- 방문 주기 2주 1회 고정 (매주 방문은 시간 낭비)

## 위너 선정 기준 (Week 4 기준)

다음 조건을 모두 충족하는 상품을 위너로 확정:

- 2주 내 판매 3건 이상
- 광고 ROAS 300% 이상
- 리뷰 3개 이상 확보

위너 확정 후 → 광고 예산 2배 확대 + 상세페이지 전면 리뉴얼 + 쿠팡 추가 등록
