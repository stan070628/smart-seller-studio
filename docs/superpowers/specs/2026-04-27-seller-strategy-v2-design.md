## 2. 전략 구조 — 3채널 + 단계적 사입

### 2.1 채널별 매출 분배 (Week 12 기준)

| 채널 | 비중 | 월매출 목표 | 역할 |
|---|---|---|---|
| 쿠팡 로켓그로스 | 50% | 500만원 | 메인 — 노출+회전, 위너 사입 집중 |
| 쿠팡 일반판매 | 25% | 250만원 | 사입 전 위탁 검증, 그로스 미진입 카테고리 |
| 네이버 스마트스토어 | 25% | 250만원 | 안전망 + 검색 노출, 동일상품 동시 등록 |

### 2.2 단계적 사입 전환 (채널 핵심 메시지)

```
Week 1~3:  도매꾹 위탁 80~100개 SKU 등록 → 빠른 노출·데이터 수집
Week 4:    위너 5~10개 선별 → 1688 첫 발주 (개당 50~100개)
Week 5~6:  1688 입고 + KC인증/상표권 검수 → 로켓그로스 입고
Week 7~9:  광고 위너 집중, 2차 사입 (개당 100~200개)
Week 10~12: 위너 스케일 사입 + 신규 시드 키워드 사이클
```

### 2.3 카테고리 회피 리스트

❌ **절대 회피**
- KC인증 필수: 전기/전자(어댑터·보조배터리), 유아용품, 식품, 유아의류
- 상표권 위험: 캐릭터(디즈니/포켓몬 등), 유명브랜드 유사 디자인
- 시즌 한정: 크리스마스/설/추석 한정 (재입고 불투명)
- 부피 큰 상품: 50cm 변 초과 (그로스 보관료 폭탄)

✅ **우선 카테고리**
- 생활용품/수납/주방소품
- 사무/문구
- 반려동물 용품 (사료 제외)
- 차량용품 (전자기기 제외)
- 인테리어 소품

## 4. 광고 예산 운영

자본 250만원은 **운영자금**. Week 7부터 매출 마진 50%를 광고비로 재투자.

| 주차 | 일 광고비/채널 | 주간 광고비 | 누적 자본 광고비 | 자본/매출 |
|---|---|---|---|---|
| Week 1 | 0 (셋업) | 0 | 0 | — |
| Week 2 | 1만원 × 3 | 21만원 | 21만 | 자본 |
| Week 3 | 1.5만 × 3 | 31만원 | 52만 | 자본 |
| Week 4 | 2만 × 3 | 42만원 | 94만 | 자본 |
| Week 5 | 2.5만 × 3 | 53만원 | 147만 | 자본 |
| Week 6 | 3만 × 3 | 63만원 | 210만 | 자본 |
| Week 7 | 4만 × 3 | 84만원 | 자본 잔여 40만(누적 250 한도 도달) + 매출재투자 44만 | 자본 종료, 매출 재투자 시작 |
| Week 8~9 | 5만 × 3 | 105만 × 2 = 210만 | 매출 재투자 210만 | 매출 |
| Week 10~12 | 6~7만 × 3 | 약 130만 × 3 = 390만 | 매출 재투자 390만 | 매출 |

**채널별 분배 원칙**
- 쿠팡 그로스 50% / 쿠팡 일반 25% / 네이버 25%
- ROAS 200% 미만 채널은 즉시 예산 30% 삭감

## 6. 안전장치 — SmartSellerStudio 활용

### 6.1 회피 리스트 자동 필터 (P0, 신규)
- `src/lib/sourcing/blocklist.ts` — KC인증 필수 카테고리 ID + 위험 키워드 사전 등록
- 도매꾹 검색 결과 자동 제외
- 1688 발주 전 최종 컨펌 게이트

### 6.2 상표권 사전 검색 (P1, 신규, KIPRIS API 연동)
- 환경변수 `KIPRIS_API_KEY` 활용 → 위너 1688 발주 전 상품명/디자인 자동 검색
- 등록 상표 발견 시 빨간 배너 + 발주 차단
- 위치: `src/app/api/trademark/precheck` + `src/lib/kipris/`

### 6.3 아이템위너 모니터링 (기존 winner-dashboard 활용)
- 매일 자동 체크, 위너 빼앗김 시 알림 + 자동 전략 제안 (가격 조정 / 옵션 분리)

### 6.4 1688 사입 체크리스트 (P1, 신규)
- 채널 영상 "회송 당합니다" 3편(포장/사이즈/바코드) 기준 체크리스트
- SKU별 입고 전 PDF 자동 생성
- 위치: `src/components/sourcing/InboundChecklist.tsx`

### 6.5 마진 계산기 (P2, 신규)
- 1688 가격 + 관세 + 배송 + 그로스 수수료 + 보관료 = 실 마진 산출
- 위탁 마진 vs 사입 마진 비교 → 사입 전환 임계값 자동 계산
- 위치: `src/components/sourcing/MarginCalc.tsx`
