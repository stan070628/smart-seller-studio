# Reference Images

레퍼런스 상품 상세페이지 이미지를 이 폴더에 저장하세요.

## 사용법

1. 분석하고 싶은 상품 상세페이지 이미지를 이 폴더에 저장 (JPG/PNG/WEBP)
2. Claude Code에서 ux-ui 에이전트에게 요청:
   - "reference 폴더 분석해서 HeroTemplate 업데이트해줘"
   - "reference 이미지 참고해서 전체 템플릿 스타일 개선해줘"

## 폴더 구조 예시

```
reference/
├── README.md
├── hero/          # 히어로 섹션 레퍼런스
├── pain_point/    # 불편함 공감 섹션 레퍼런스
├── detail/        # 상품 상세 섹션 레퍼런스
└── cta/           # 구매 유도 섹션 레퍼런스
```

이미지 파일을 하위 폴더 없이 바로 저장해도 됩니다.
